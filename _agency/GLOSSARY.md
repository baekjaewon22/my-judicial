# 도메인 용어집

> 코드·커밋·회의에서 혼동되는 용어를 여기서 일관되게 정의.
> 새 용어 발견 시 즉시 추가. 애매한 용어는 대표님 확인.

## 공통 — 권한·사용자

| 용어 | 정의 | 코드 표현 |
|---|---|---|
| 대표/관리자 | 모든 권한 | `role = 'admin'` |
| 총무담당 | 매출·정산 실무자 | `role = 'accountant'` |
| 총무보조 | 입력 전담, 변경 승인 필요 | `role = 'accountant_asst'` |
| 열람자 | 조회만 가능 | `role = 'viewer'` |
| 승인 대기 | 🔸 액션이 승인을 기다리는 상태 | `pending_approvals` 테이블 |

## 공통 — 매출·정산

| 용어 | 정의 |
|---|---|
| 매출 등록 | 거래 발생을 시스템에 최초 기록하는 행위 |
| 정산 | 일정 기간 매출의 집계·분배·입금 확인 |
| 입금일 매칭 | 카드결제 승인일과 실제 입금일을 연결 |
| 차감 | 보고서 재제출·환불 등으로 기록된 매출을 감액 |

## 권리분석 (마이옥션 인사이드 전용)

| 용어 | 정의 |
|---|---|
| 경매 물건 | 법원 경매 대상 부동산 |
| 물건분석보고서 | 물건의 권리관계·리스크·수익성을 정리한 A4 HTML 문서 |
| 과밀억제권역 | 수도권정비계획법상 과밀 억제 대상 지역 (취득세 중과 등 영향) |
| 배당순위 | 경매 낙찰금을 채권자에게 분배하는 법정 순서 |
| 대항력 | 전입신고 + 점유로 임차인이 취득하는 보호 요건 |
| 전입세대 | 해당 물건지에 주민등록된 세대 현황 |
| 취득세 분석 | 낙찰가 기준 취득세·부대비용 시뮬레이션 |
| 사건번호 | 법원이 경매 사건에 부여하는 식별자 (예: 2024타경12345) |

## 알림톡·메시징

| 용어 | 정의 |
|---|---|
| 알림톡 | 카카오톡 채널 기반 정보성 메시지 (NCP Biz Message 경유) |
| 승인 템플릿 | NCP/카카오에 등록·심사 완료된 메시지 포맷. 승인된 것만 발송 가능 |
| 템플릿 ID | NCP 시스템 내 템플릿 식별자 |
| 치환 변수 | 템플릿의 `#{변수명}` 으로 표시되는 동적 치환 지점 |
| 대체 발송 | 알림톡 실패 시 SMS 로 폴백 |
