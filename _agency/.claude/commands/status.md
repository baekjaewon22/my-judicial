# /status — 진행 상황 대시보드

프로젝트의 현 상태를 한눈에 정리합니다.

## 수집 정보

### 1. 목표·마일스톤
- `PROJECT.md` 에서 목표 체크박스 상태
- 다가오는 마일스톤 (3개까지)

### 2. 백로그
- `BACKLOG.md` 에서:
  - in_progress 항목 전체
  - blocked 항목 전체
  - todo 상위 5건 (우선순위 순)

### 3. 최근 회의
- `MEETING-LOG.md` 최근 3건
- 액션 미반영 회의가 있으면 경고

### 4. 의사결정
- `DECISIONS.md` 에서 `proposed` 상태 ADR
- 최근 2주 `accepted` ADR

### 5. Git
```bash
git status --short
git log --oneline -5
git diff --stat HEAD~5..HEAD
```

## 원칙

- **출력은 한 화면**에 들어오도록 압축
- 숫자·상태 중심, 장문 설명 금지
- "대표님이 지금 뭘 해야 하는지" 결론 섹션 포함
- 2분 이내 완료
